# OSF Registration: 10-Minute Walk-Through

Do this AFTER you've edited and signed FREEZE_v1_DRAFT.md. Nothing pricing-related runs until this is registered.

1. Log in at osf.io (account exists; token already in env for later API use).
2. Create a new PRIVATE project. Title: "Small business asking-price multiples by service category, 2019-2026: a descriptive study" (bland on purpose; project titles can become visible, so no "AI" and no "Repricing Main Street" in it).
3. Upload the signed freeze doc (export FREEZE_v1_DRAFT.md to PDF or paste contents) into the project files.
4. From the project: Registrations → New registration → choose **Open-Ended Registration** (the simplest template).
5. In the registration summary field, paste the full text of the signed freeze doc (or attach it and state "specification is the attached signed document, FREEZE v1").
6. On the final screen choose **Embargo**, set the end date to **2026-08-05**. The registration is timestamped immediately; contents stay private until the embargo lifts (or until you end the embargo early at launch).
7. Submit. Copy the registration URL and DOI.
8. Paste the URL + DOI + timestamp into a new file `07_freeze/OSF_RECORD.md` (or into chat and I'll file it). The frozen doc then gets renamed FREEZE_v1_SIGNED.md in the repo, and that exact text never changes again.

That's Gate 0's first leg. After this: panel wave 1 and the Wayback audit are allowed to run.
